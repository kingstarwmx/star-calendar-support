## 常见问题（FAQ）

**1. 看不到系统日历里的事件？**  
请在系统设置中允许日历权限：  
`设置 → 隐私与安全性 → 日历 → 星星日历 → 允许访问`

**2. 语音输入不可用或没有反应？**  
请检查权限：  
`设置 → 隐私与安全性 → 麦克风/语音识别 → 星星日历 → 允许`

**3. 无法定位或地点搜索为空？**  
请检查定位权限：  
`设置 → 隐私与安全性 → 定位服务 → 星星日历 → 使用App期间`

**4. 提醒没有准时弹出？**  
- 确保系统通知已允许  
- 检查是否开启“专注模式”或“勿扰模式”  
- 在系统设置中允许通知：  
  `设置 → 通知 → 星星日历 → 允许通知`

**5. 日历内容没有更新？**  
尝试下拉刷新，或重启 App。

## 故障排查步骤

1. 更新到最新版本  
2. 重启应用  
3. 检查系统权限  
4. 重新安装 App（数据可能丢失，请谨慎）
