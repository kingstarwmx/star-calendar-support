## FAQ

**1. I can’t see events from the system calendar.**  
Please allow Calendar access:  
`Settings → Privacy & Security → Calendars → Star Calendar → Allow`

**2. Voice input doesn’t work.**  
Please allow Microphone and Speech Recognition permissions:  
`Settings → Privacy & Security → Microphone/Speech Recognition → Star Calendar → Allow`

**3. Location search is empty or current location fails.**  
Please allow Location access:  
`Settings → Privacy & Security → Location Services → Star Calendar → While Using the App`

**4. Reminders don’t fire on time.**  
- Make sure notifications are enabled  
- Check Focus/Do Not Disturb settings  
- Enable notifications:  
  `Settings → Notifications → Star Calendar → Allow Notifications`

**5. Calendar doesn’t refresh.**  
Pull to refresh or restart the app.
